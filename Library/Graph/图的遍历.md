# 图的遍历

## DFS

分为递归和非递归两种实现方法，递归更加简单，非递归使用了一个栈来记录遍历顺序，可以减少时间和空间的消耗。

递归实现:
```cpp
void DFS_recursive(Graph &graph, int startVertex, int visited[]) {
    doSth(startVertex);
    visited[startVertex] = 1;
    for (int i = 0; i < graph.NumVertex; i++)
        if (!visited[i] && graph.G[i][startVertex] < MaxWeight)
            DFS_recursive(graph, i, visited);
}
```

非递归实现:
```cpp
void DFS_nonrecursive(Graph &graph, int startVertex) {
    stack<int> s;
    int i, currentVertex;
    int visited[MaxVertexNum] = {0};
    s.push(startVertex);
    visited[startVertex] = 1;
    while (s.size()) {
        currentVertex = s.top();
        doSth(currentVertex);
        s.pop();
        for (i = graph.NumVertex - 1; i > 0; i--)
            if (!visited[i] && graph.G[i][currentVertex] < MaxWeight) {
                s.push(i);              
                visited[i] = 1;
            }
    }
}
```

## BFS

BFS使用递归实现与非递归实现思路相似，但是浪费资源，所以使用非递归来实现。

```cpp
void BFS(Graph &graph, int startVertex) {
    queue<int> s;
    int i, currentVertex;
    int visited[MaxVertexNum] = {0};
    s.push(startVertex);
    visited[startVertex] = 1;
    while (s.size()) {
        currentVertex = s.front();
        doSth(currentVertex);
        s.pop();
        for (i = 0; i < graph.NumVertex; i++)
            if (!visited[i] && graph.G[i][currentVertex] < MaxWeight) {
                s.push(i);              
                visited[i] = 1;
            }
    }
}
```
